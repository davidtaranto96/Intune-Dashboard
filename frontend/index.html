ℹ️ About this tool

The Intune Compliance Dashboard provides real-time visibility into Microsoft Intune managed devices and Entra ID security for JBKnowledge. Monitor compliance policies, conditional access enforcement, sign-in activity, and take action on non-compliant devices. Requires IT Administrator permissions.
