ℹ️ About this tool

The Intune Compliance Dashboard provides real-time visibility into Microsoft Intune managed devices and Entra ID security for JBKnowledge. Monitor compliance policies, conditional access enforcement, sign-in activity, and take action on non-compliant devices. Requires IT Administrator permissions.

    📄 What's New

    v3.1.0
    April 2026 — UX & Bilingual Update
    NewBilingual Support (EN / ES) — Toggle between English and Español (Argentina) from both the login page and the main header. Preference saved automatically.
    NewClickable Security Intelligence Badges — Filter Risky Users by High/Medium/Low/At Risk and Defender Alerts by severity. Active badge highlighted with border and shadow.
    NewRisky Users → User Detail Modal — Click any user row in the Risky Users card to open their full sign-in history, stats, and action buttons.
    FixNONE badge on Risky Users — Now correctly shows the risk state (AT RISK, CONFIRMED COMPROMISED) instead of "NONE" when risk level is unresolved.
    ImproveSummary Cards — Single Row — Sign-in counter cards (Total, Successful, Blocked, CA Blocked, MFA, Risky) now fit in one compact horizontal row.
    ImproveSecure Score Card — Now clickable; opens Microsoft Secure Score portal in a new tab.
    v3.0.0
    April 2026 — Security Actions & Intelligence Update
    NewQuick Security Actions — Block User, Force Password Reset, Revoke Sessions, and Confirm Compromised buttons now available directly from sign-in detail modal and Smart Insights cards.
    NewSecurity Intelligence Panel on Dashboard — Microsoft Secure Score gauge, Risky Users from Identity Protection, and Defender Alerts, all in real-time.
    NewSmart Insights v2 — risk badges (CRITICAL/HIGH/MEDIUM/LOW), top failure reason, block rate %, configurable top 5/10/15/20, dynamic security recommendations panel.
    ImproveSign-in Time Range now works correctly — fixed pagination and ISO timestamp format, supports up to Top 10000 records over 30 days.
    ImproveLayout — Smart Insights moved above filter bar; filter buttons now sit directly above the activity table they control.
    v2.5.0
    April 2026 — Security & Analytics Update
    NewSign-in Analytics Charts on Dashboard — see where your users log in from (by country) and why sign-ins are being blocked (wrong password, MFA, CA policy, etc.)
    NewSecurity Overview on Dashboard — 4 cards showing blocked sign-ins, risky sign-ins, account lockouts, and CA-blocked. Click any card to see the affected events.
    NewSign-in Detail Modal — click any row in Sign-in Logs for full details: IP, location, device trust type, risk level, MFA method, all CA policies applied/blocked, error code, and failure reason.
    NewMixed Alerts — Alerts tab now shows both device compliance alerts and sign-in security alerts (blocked, risky, lockouts, repeated failures) in one unified list.
    NewEmail Notifications — get alerted by email when issues are detected. Configure recipients, sender, and alert types in the Alerts tab. Requires Mail.Send permission.
    ImproveUsers Without Device list — collapsed to 5 rows by default with a search box. Click "Show all" to expand the full list.
    ImproveSign-in Logs table — blocked rows highlighted red, risky rows orange. Added device trust type column. Row highlighting makes issues stand out at a glance.
    ImproveDashboard Config — new toggles for Security Overview and Sign-in Analytics Charts sections.
    FixDashboard freeze on tab switch — all charts (compliance, OS, sign-in) now re-render correctly when you return to the Dashboard tab.
    v2.4.0
    March 2026 — Access Control Update
    NewDomain restriction — only @jbknowledge.com accounts can access the dashboard. Others see "Access Denied" with option to switch accounts.
    FixLogin loop fixed — Sign Out properly clears the session. Login always prompts for credentials (prompt=login).
    ImproveSync device error — clearer message when PrivilegedOperations.All permission is missing instead of a generic 403 error.
    v2.3.0
    March 2026 — Dashboard & Reports Update
    NewConfigurable Dashboard — toggle visibility of each section and choose chart style (donut / pie / bar).
    NewCompliance by Platform chart — stacked bar showing compliant vs non-compliant per OS.
    NewUser Enrollment Coverage — shows which Entra ID users have no Intune device enrolled.
    NewClickable Alerts — click any alert to see exactly which devices are affected.
    ImproveReports tab redesigned — dropdown selects for type, compliance filter, OS filter, and record count. CSV export for devices, policies, sign-ins, and user coverage.
